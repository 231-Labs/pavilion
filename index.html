PAVILION

3D NFT Gallery

Transforming Kiosks into Immersive Galleries

Sui Blockchain Three.js Move Language Web3

Problem & Motivation

Current Limitations of 3D NFTs

  • Most 3D NFTs only store metadata on-chain
  • Lack of immersive exhibition experience
  • No intuitive display and interaction
  • Collectors struggle to manage digital assets effectively

We need a visible, tangible, adjustable solution

Pavilion Solution

Core Concept

  • Transform Kiosks into 3D exhibition spaces
  • Real-time loading and display of NFT assets
  • Support position, rotation, and scale adjustments
  • Permanent scene configuration storage on blockchain
[Pavilion 3D Gallery Screenshot]

Technical Architecture

Frontend Technology

  • Three.js - 3D Rendering Engine
  • React - User Interface
  • Next.js - Full-stack Framework
  • TypeScript - Type Safety

Blockchain Technology

  • Sui - High-performance Blockchain
  • Move - Smart Contract Language
  • Kiosk - NFT Management Protocol
  • Dynamic Fields - Scene Storage

Demo Flow

Three-Step Pavilion Creation

  1. Create Pavilion - Create new Kiosk or use existing one
  2. Place NFTs - Load 3D models from wallet or Walrus
  3. Adjust & Save - Freely adjust positions and permanently save on-chain

All operations through modernist glass interface
